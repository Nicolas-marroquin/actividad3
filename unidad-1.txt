Sistema binario
Es un lenguaje el cual utilizan las computadoras para procesar las instrucciones que ingresamos en 
código el cual nosotros si entendemos. 
Se represente con 2 dígitos “1” y el “0” para convertir números decimales a binarios hay dos métodos: 

1. Divisiones utilizando el residuo: este método consiste en dividir una cantidad hasta tener un residuo 
igual a 0 o 1 guardamos el residuo
Luego realice una nueva división utilizando el cociente de la división anterior como dividiendo repita 
el paso hasta que el cociente sea 0 o 1 y guardamos el residuo 
Cuando el cociente sea igual a cero  o ya no se pueda seguir dividiendo con el último cociente procedemos 
a construir con los residuos guardados el número binario.

2. potencia de 2:

Lógica 
Gracias a la lógica podemos determinar si un razonamiento es válido o no válido, verdadero o falso 
Una proposición lógica es un enunciado al que se le puede asignar un valor verdadero o falso 
Por ejemplo 
Los pandas son amigables, El cielo es azul, La noche es oscura, El agua es fría 

Algunas proposiciones son siempre verdaderas otras falsas y otras dependen de una condición extra: 

Por ejemplo los pandas no siempre son amigables depende mucho del temperamento en el que este se 
encuentre de lo contrario a veces son agresivos.

El valor verdadero o valido de una proposición casi siempre es sujeto a una condición por eso siempre
 es importante colocar contexto alas preposiciones 

La conjunción se utiliza para unir dos proposiciones verdaderas para obtener un resultado verdadero 

Por ejemplo mi perro es alto y es juguetón 

La negación como su nombre lo indica Se utiliza para negar una proposición lógica
Por ejemplo 
El carro es rojo 
El carro no es rojo 
La disyunción Se utiliza Para conectar dos preposiciones lógicas donde cualquiera de las dos debe ser verdadera.
Por ejemplo 
Mi casa es grande o mi casa es pequeña
